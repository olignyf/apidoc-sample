# apidoc-sample

This is a minimal Express project with apidoc.

## Usage

1. Install dependencies:
   ```
npm install
   ```
2. Start the server:
   ```
npm start
   ```
3. Generate API documentation:
   ```
npm run build
   ```
4. Open `apidoc/index.html` in your browser to view the generated docs.
